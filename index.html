Side Projects

FlashCard: an iOS app that lets me study wherever I am, whenever I want. New cards are shown by swiping to the left or selecting "correct" or "incorrect". Cards are flipped by tapping once on the card's text. The cards are separated into piles based on correctness in order to quickly identify which cards need to be studied.
Language: Objective-C / Xcode


SoundMap: This script uses SoundCloud's API to retrieve a list of the most recent SoundCloud uploads containing location data. This data is then geocoded and a marked on a map using the Google Maps API.
Language: JavaScript / SoundCloud API / Google Maps API

SimonSays: Recreated a memorization game that I played often as a kid.
Language: Objective-C / Xcode
